Instructions

To type directly with the computer keyboard:

  • Use the capital letters to type the aspired characters: K T P J G D B or kh th ph jh dh bh
  • Use the capital letter N or ng
  • Type a space after the consonant to get the final consonant: k l m p n t N I
  • Type M for the anusvara  ꯪ

Copy [Ctrl]+[C] & Paste [Ctrl]+[V]
