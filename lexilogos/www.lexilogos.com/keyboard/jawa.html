Note

Click on the virama, called pangkon in Javanese (diacritic in the center) to delete the inherent vowel a: the virama is needed between two consonants.

Instructions

To type directly with the computer keyboard:

  • Use the capital to type the letters subscribed with a dot below: T, Th, D, Dh, N, L, S
  • Type G or ng for ṅ & J for ñ
  • Type ç or z or sh for ś
  • Type aa, ii, uu (or A, I, U) for the long vowels ā, ī, ū
  • Type -r and -ri for ri and rī
  • Type -l and -li for li and lī
  • Type M for ꦁ (anusvara) & MM for ꦀ (candra bindu)
  • Type H for ꦃ (visarga)
  • Type R for ꦂ (layar)
  • Type = for ꦳ (nukta)
  • Type / or | for ꧈ (danda)
