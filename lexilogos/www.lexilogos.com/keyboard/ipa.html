Instructions

Type the diacritical mark after the character.


To type directly with the computer keyboard:

  • Type = (and several times =) to get a closed character (for example o= to get ɔ)
  • Type ' to get the principal accent
  • Type : for a long vowel
  • Type N to mark a nasal
