Instructions

To type directly with the computer keyboard:

  • For the emphatic letters, type the capital letters: H, H=, S, S= T, T=
  • Type t= for ṯ and t== for ṱ
  • Type d= for ḏ
  • Type x (or H=) for ḫ
  • Type s=, s== (or s1, s2) for š, ś
