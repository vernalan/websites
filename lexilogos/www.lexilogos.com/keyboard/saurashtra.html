Instructions

To type directly with the computer keyboard:

  • Use the capital to type the letters subscribed with a dot below: T, Th, D, Dh, N, L, S
  • Type G for ng & J for ñ
  • Type ç or z or sh for ś
  • Type aa, ii, uu, ee (or A, I, U, E) for les voyelles longues ā, ī, ū, ē
  • Type -r and -ri for ri and rī
  • Type -l and -li for li and lī
  • Type -h for ꢴ (hāra or upakshara)
  • Type M for ꢀ (anusvara) & MM for ꣅ (candra bindu)
  • Type H for ꢁ (visarga)
  • Type / or | for ꣎ (danda) and twice for a double danda
