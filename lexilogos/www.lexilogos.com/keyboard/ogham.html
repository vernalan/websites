Instructions

To type directly with the computer keyboard:

  • Type = to add a stroke
  • Type a capital letter E, O, U, I, A for: éa, ói, úi, ío, ae
  • Type > and < for the feathers (marks beginning and end of Ogham text)
  • Type - or _ for a space mark

Copy [Ctrl]+[C] & Paste [Ctrl]+[V]
Terms:

fid (plural: feda): character ("tree")
aicme (plural: aicmí): group ("family") of 5 characters
forfeda: additional characters (mainly in manuscripts)
