Note

Click on the ceung mark ្ to get the following letter in a subscript form (the foot of the consonant) example:
ត + ្ + វ = ត្វ

Instructions

To type directly with the computer keyboard:

  • Use the capital to type the letters subscribed with a dot below: T, Th, D, Dh, N, S
  • Type go & gho for ko & kho / do & dho for to & tho / bha for pha
  • Type G to get ng & J for ñ
  • Type z or ç or sh for ś
  • Type aa, ii, uu for the long vowels ā, ī, ū
  • Type w and ww for the vowels ə, ə̄
  • Type -r and -ri for ri and rī
  • Type -l and -li for li and lī
  • Type M, H, Q to get ំ ះ ៈ
  • Type / or | and || for the full stop ។ ៕
  • taper M, H, Q pour ។ ៕
