Instructions

To type directly with the computer keyboard:

  • For characters specific to Rohingya, type an uppercase: R, V, Y
  • To add a diacritic, type an apostrophe
  • Type AA for ā
  • For the nasal vowels (second line), type aa, oo, ii, ee, uu, OO

Copy [Ctrl]+[C] & Paste [Ctrl]+[V]
