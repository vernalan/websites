Instructions

To type directly with the computer keyboard:

  • Type a consonant to get ə (by default)
  • Type e for ä and ee (or é) for ē
  • Type aa for ʷa
  • Type a capital letter H, T, C, P, S for ḥ   ṭ   č̣   p̣   ṣ
  • Type ss (or sh), c (or tt), zz (or zh), j (or dd) for š   č   ž   ǧ
  • Type a capital letter N (or n=) for ñ
  • Type a capital letter X (or h=) for ḫ
  • Type sz, tz (or s=, S=) for ś   ṣ́
  • Type < or the apostrophe ['] for the glottals or the lower case vowels
  • Type > or the quotation marks ["] for the pharyngals or the capital vowels
