Instructions:

To type directly with the computer keyboard:

  • Type aa or â to have ā, -a to add the hamza
  • For the letters with a diacritical mark, add an apostrophe
    example: t', j' (or c), H' (or x), d', s', S', T', G', k' (or g)
    or two apostrophes: j'', z'', s''
  • Type tt, dd, rr for the Urdu and Torwali special characters

Copy [Ctrl]+[C] & Paste [Ctrl]+[V]


• SIL: if the characters don't appear correctly, Download & install the font Scheherazade
