Instructions

To type directly with the computer keyboard:

  • Add an apostrophe: t', p', k', c', r' (or t=, p=, k=, c=, r=)
  • Type = to add a diacritic: z= (or zh), s= (or sh), j= (or jh)
  • Type c= (or c'), c== (or ch), c=== (or ch' or tc)
  • Type e= ou ë
