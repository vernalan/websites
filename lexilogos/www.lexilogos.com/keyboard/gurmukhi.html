Note

Click on the virama (diacritic in the center) to delete the inherent vowel a.

Place a virama between two consonants, also to form a ligature.
example: n + virama + h for ਨ੍ਹ​



Instructions

To type directly with the computer keyboard:

  • Use the capital to type the letters subscribed with a dot below: T, Th, D, Dh, N, R, Rh, L, S
  • Type G for ng & J for ñ
  • Type z or ç or sh for ś
  • Type aa, ii, uu (or A, I, U) for the long vowels ā, ī, ū
  • Type M for ਂ (bindi) & MM for ਁ (adak bindi)
  • Type H for ਃ (visarga)
  • Type = for ਼ (nukta): dot below
  • Type ee for ੰ
  • Type x to form a half consonant; example: type xr pour ‍੍ਰ
  • Type / ou | pour । (punctuation)
