Instructions

To type directly with the computer keyboard:

  • Type aa or â to get ā
  • For the letters with a diacritical mark, add an apostrophe,
    example: t', j' (or c), H' (or x), d', s', S', T', G', k' (or g)
  • Type tt, dd, ll, nn, rr for the special characters of the Punjabi
  • Type h' pour ʰ (for the aspirate consonants)
  • Type - for the hamza
  • Type y-- to get ئ
  • Type &a, &u, &i for the diacritics and && to mark a long consonant
