Instructions

To type directly with the computer keyboard:

  • Type = to change the character of the same phoneme: kh=, th=, ph=, f=, s=, h=
    the first character is a high tone (example: s), the second one a low tone (example: s=)
  • Type aa, ii, uu, UU for the long vowels: ā, ī, ư̄, ū
  • Type an apostrophe to add a tone and several times to change the tone.
