Instructions

To type directly with the computer keyboard:

  • To add a diacritical mark, type an apostrophe; example: t', H' (or x), d', s' (or ş), G' (or ğ), k' (or ñ)
  • Type â or aa for ā
  • Type / to avoid the ligature between two characters
