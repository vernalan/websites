Instructions

To type directly with the computer keyboard:

  • Type the capitals O for ɔ, E for for ɛ
  • Type = after the vowel for a nasal ◌̃
  • Type B for ɓ, D for ɗ, J for ʒ
  • Type dh for ð, th for θ, sh for ʃ
  • Type ny (or gn or N) for ɲ (like "canyon") and ng (or G) for ŋ (like "king")

If the characters are not displayed, download & install the font Noto Sans Vai
