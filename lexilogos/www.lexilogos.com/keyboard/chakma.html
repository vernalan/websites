Note

Cliclk on the virama 𑄳 (diacritic in the center) to delete the inherent vowel ā (aa)

example: mh 𑄟𑄳𑄦 is typed with 3 characters: 𑄟 𑄳 𑄦 (m + virama + h)

Instructions

To type directly with the computer keyboard:

  • Type + for the virama (to combine two consonants)
  • Type k for 𑄇𑄴, ka for 𑄇𑄧, kaa for 𑄇
  • Use the capital to type the letters subscribed with a dot below: T, Th, D, Dh, N
  • Type G for ng & J for ñ
  • Type aa, ii, uu (ou A, I, U) for the long vowels ā, ī, ū
  • Type M for 𑄁 (anusvara) & MM for 𑄀 (candra bindu)
  • Type H for 𑄂 (visarga)
  • Type / or | for 𑅁 (danda) and twice for a double danda
