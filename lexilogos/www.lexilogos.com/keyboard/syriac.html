Instructions:

To type directly with the computer keyboard:

  • Use the capitals: H   T   S for ḥ   ṭ   ṣ
  • Type c for sh
  • Type a for ʾ
  • Type - for ʿ

Copy [Ctrl]+[C] & Paste [Ctrl]+[V]

Notes:

The Syriac is written from right to left.

Phoenician, Aramaic, Hebrew or Arabic characters can be copied in the frame.
