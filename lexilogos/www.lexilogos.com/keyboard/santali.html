Instructions

To type directly with the computer keyboard:

  • Type aa (or A) for ā
  • Use the capital to type the letters with a dot: T, D, N, R
  • Type G or ng for ṅ & J for ñ
  • Type M for ṃ
  • Type MM for ᱸ (nasalisation)
  • Type ' for ᱽ
  • Type = to add a diacritic
