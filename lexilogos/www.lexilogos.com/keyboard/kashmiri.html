Instructions

To type directly with the computer keyboard:

  • Type twice a vowel to get a long vowel; example: aa to get ā
  • Type w to get the diacritic of the vowel a
  • For the letters with a diacritical mark, add an apostrophe; example: t', j' (or c), H' (or x), d', s', S', T', G', k' (or g)
  • Type tt, dd, rr for the special characters (like in Urdu)
  • Type h' for ʰ
  • Type - for the hamza
