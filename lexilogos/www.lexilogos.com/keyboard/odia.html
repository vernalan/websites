Instructions

To type directly with the computer keyboard:

  • Use the capital to type the letters subscribed with a dot below: T, Th, D, Dh, N, R, Rh, S
  • Type G for ng & J for ñ
  • Type z or ç or sh for the character ś
  • Type aa, ii, uu (or A, I, U) for the long vowels ā, ī, ū
  • Type -r and -ri for ri and rī
  • Type -l and -li for li and lī
  • Type M for ଂ (anusvara) & MM for ଁ (candra bindu)
  • Type H for ଃ (visarga)
  • Type = for ଼ (nukta): dot below
  • Type '' (2 apostrophes) for ଽ (avagraha)
  • Type / or | for । (danda) and twice for a double danda
