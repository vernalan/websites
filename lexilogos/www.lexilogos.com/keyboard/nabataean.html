Instructions:

To type directly with the computer keyboard:

  • Use the capitals: H   T   S for ḥ   ṭ   ṣ
  • Type c for š
  • Type a for ʾ
  • Type - for ʿ
  • For the final consonants, type an escape after the consonant

Download the font Noto Sans Nabataean


Copy [Ctrl]+[C] & Paste [Ctrl]+[V]

Note:

The Nabataean is written from right to left.
